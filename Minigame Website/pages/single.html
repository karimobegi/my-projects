Projet de site web de minijeux

1. Guess My Number (Devine mon nombre)

Tutoriel:

Générez un nombre aléatoire entre 1 et 20. Invitez l'utilisateur à entrer un nombre. Comparez le nombre entré avec le nombre généré. Fournissez des indices (plus grand, plus petit) et réduisez le nombre de vies. Répétez les étapes 2-4 jusqu'à ce que l'utilisateur trouve le nombre ou épuise ses vies.

2. Memory Card Game (Jeu de mémoire)

Tutoriel:

Créez un tableau de cartes (4x4 dans ce cas). Associez chaque paire de cartes avec une valeur identique. Mélangez les cartes pour qu'elles apparaissent aléatoirement. Affichez les cartes au joueur. Laissez le joueur choisir deux cartes à la fois. Vérifiez si les cartes sont une correspondance. Si les cartes correspondent, gardez-les affichées. Sinon, retournez-les. Répétez jusqu'à ce que toutes les paires soient trouvées ou que le joueur épuise ses vies.

3. Guess the Word (Devine le mot)

Tutoriel:

Préparez une liste de mots possibles. Choisissez un mot au hasard. Affichez des traits soulignés pour chaque lettre du mot. Invitez l'utilisateur à deviner une lettre ou le mot complet. Vérifiez si la lettre est dans le mot. Si oui, révélez les positions. Réduisez le nombre de tentatives et répétez les étapes 4-5. Si l'utilisateur devine correctement le mot ou épuise ses tentatives, terminez le jeu.
